# Professional Portfolio

A personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: Works seamlessly on desktop and mobile.
- **Interactive Elements**: Animations, floating navigation, and dynamic content.
- **Sections**:
  - Hero Section
  - About (Education, Experience, Skills)
  - Projects (Showcase with links)
  - Certificates (Categorized with external links)
  - Contact Form

## Technologies Used
- HTML5
- CSS3 (Custom variables, Flexbox, Grid)
- JavaScript (DOM manipulation)
- FontAwesome Icons
- Google Fonts (Poppins)

## Setup
Simply open `index.html` in your web browser to view the portfolio.
